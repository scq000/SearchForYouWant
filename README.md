#Description
A search tools for downloading based in BT or [BaiduPan](https://pan.baidu.com).

#Get Start 使用方法
1. 安装python和pip以及依赖的工具库
以os x 系统为例
```
brew install python
brew install pip
sudo pip install bs4
sudo pip install requests
```
2. 命令行中使用
```
python search.py keyword
```
keyword为你想要搜索的资源关键字
